Вопрос: Объясните механизм внимания (Attention) в архитектуре трансформеров. Как работает self-attention и почему он важен?
----------------------------------------------------------------------
Тип ответа: excellent
Ответ: Механизм внимания в трансформерах позволяет модели динамически определять важность различных частей входной последовательности при обработке каждого элемента. В self-attention для каждого токена создаются три вектора: Query (запрос), Key (ключ) и Value (значение) через линейные преобразования с весовыми матрицами W_Q, W_K, W_V.

Scaled dot-product attention вычисляется по формуле: Attention(Q, K, V) = softmax(Q·K^T / √d_k) · V. Масштабирование на квадратный корень из размерности ключей d_k критически важно — без него при больших d_k скалярные произведения становятся слишком большими, что приводит к затухающим градиентам в softmax.

Multi-head attention использует h параллельных голов внимания, каждая из которых учится фокусироваться на разных аспектах данных. Результаты конкатенируются и проецируются через линейный слой.

Ключевые преимущества перед RNN/LSTM: полная параллелизация (все позиции обрабатываются одновременно), возможность моделировать длинные зависимости без затухания градиента, интерпретируемость через визуализацию весов внимания.
